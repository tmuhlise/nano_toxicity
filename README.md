This repository contains the code and workflows for an interpretable two-stage machine learning (ML) nano-QSAR framework developed to predict the toxicity and biological effects of metal oxide nanoparticles (MONPs).
The framework integrates physicochemical descriptors, exposure conditions, and an intermediate Toxicity Class to improve prediction accuracy and biological interpretability
  Classification dataset: kona1.xlsx= [Data Curation to Develop Machine Learning Models for Assessing the Toxicity of Nanoparticles](https://jstagedata.jst.go.jp/articles/dataset/Machine_Learning-Ready_Dataset_for_Cytotoxicity_Prediction_of_Metal_Oxide_Nanoparticles/29672717)
  Regression dataset: kona.xlsx= https://www.kaggle.com/datasets/ucimachinelearning/nanoparticle-toxicity-dataset
